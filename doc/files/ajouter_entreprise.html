ajouter_entreprise.php

Formulaire d'ajout d'une entreprise

Package
\mod_stages

Tags

version

Permet à l'utilisateur d'ajouter une entreprise avec son tuteur Le tuteur est employé surpervisant l'étudiant au sein de l'entreprise

Vérification des données requises avant l'envoie de la requête pour remplir la base de données

Dû aux contraintes de clefs étrangère de la base de données, le tuteur est d'abord ajouté à la base, puis il est selectionner, puis l'entreprise est ajouter avec la clef étrangère du tuteur correspondant
